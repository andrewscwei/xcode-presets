## Usage

Install via Wget:

```sh
$ wget -qO- https://raw.githubusercontent.com/andrewscwei/xcode-presets/master/install.sh | bash
```

Export current Xcode settings:

```sh
$ git clone https://github.com/andrewscwei/xcode-presets
$ cd xcode-presets
$ ./export.sh
```
